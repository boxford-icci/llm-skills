# Feature workflow

Use this when a feature is moving from idea to implementation and ship gate.

## Suggested sequence

1. `The Product Mind`
2. `The Architect`
3. `The Designer`
4. `The Executor`
5. `The Tester`
6. `The Enforcer`

## Copy-paste prompt

```text
Use The Orchestrator to route this feature from product framing through implementation and pre-merge review. Prefer the smallest useful lineup, but default to Product Mind, Architect, Designer, Executor, Tester, and Enforcer if the work spans the full lifecycle.
```
