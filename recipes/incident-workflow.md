# Incident workflow

Use this after a regression, outage, or production surprise.

## Suggested sequence

1. `The Debugger`
2. `The Reliability Engineer`
3. `The Investigator`
4. `The Architect`

## Copy-paste prompt

```text
Use The Orchestrator to route this incident. I want root cause, reliability weaknesses, process failures, and the structural changes that would reduce recurrence risk. Keep the lineup minimal but biased toward Debugger, Reliability Engineer, Investigator, and Architect.
```
