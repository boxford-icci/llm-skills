# Example fleet review transcript

## User

```text
Run a fleet review on this checkout redesign. I need UX/UI quality, accessibility, and a final ship call.
```

## Good sequence

1. `The Designer` reviews flow, hierarchy, and UI quality.
2. `The Accessibility Engineer` reviews keyboard access, semantics, focus, and screen-reader behavior.
3. `The Enforcer` decides whether the remaining risk is shippable.

## Good synthesis shape

```text
- Design blockers
- Accessibility blockers
- Shared fixes that solve both
- Final ship / fix / rethink verdict
```
